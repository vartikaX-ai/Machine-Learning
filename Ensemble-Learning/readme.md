# Ensemble Learning

## About this Repository

This repository contains my implementation and practical project on **Ensemble Learning** using **Python** and **Scikit-learn**.

The objective of this repository is to understand how multiple machine learning models can be combined to improve prediction performance. Currently, this repository focuses on the **Bagging** ensemble technique and its practical implementation on a real-world dataset.
